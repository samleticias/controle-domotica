Sobre o Projeto

Este sistema foi criado como parte das atividades da disciplina Programação para Internet II no IFPI. Ele permite gerenciar de forma inteligente diferentes ambientes residenciais, oferecendo controle de iluminação, climatização e automação de cenas para dispositivos conectados.


Tecnologias Utilizadas

Servidores e Banco de Dados

  • Node.js: Responsável por executar o backend.
  • Express.js: Facilita a criação da API que comunica front e back.
  • PostgreSQL: Armazena informações sobre casas, cômodos e dispositivos.
  • Sequelize: Simplifica o acesso ao banco de dados usando objetos JavaScript.
  • Nodemon: Atualiza automaticamente o servidor durante alterações no código.

Interface do Usuário

  • HTML: Estrutura e conteúdo das páginas.
  • CSS: Estiliza a interface, mantendo cores e tipografia consistentes.
  • JavaScript: Adiciona interatividade e conecta com a API do backend.

Principais Funcionalidades

  • Cadastro e gerenciamento de Casas e Cômodos.
  • Controle individual de Dispositivos, com possibilidade de ligar ou desligar.
  • Criação de Cenas personalizadas, acionando múltiplos dispositivos de forma automática.
